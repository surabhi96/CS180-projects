CS180 Project: Fun with Filters and Frequencies!

This project explores 2D convolution, filtering, frequencies, and multi-resolution blending. All results shown below were implemented from scratch in Python.

Part 1: Fun with Filters

Part 1.1: Convolutions from Scratch!

Implemented convolution with four loops, then two loops, and compared against scipy.signal.convolve2d. Below is an example using a 9x9 box filter and finite difference operators.

Part 1.2: Finite Difference Operator

We computed partial derivatives in X and Y of the cameraman image, then gradient magnitude and binarized edge maps.

Part 1.3: Derivative of Gaussian (DoG) Filter

We smoothed the image with a Gaussian filter, then computed derivatives using DoG filters.

Part 2: Fun with Frequencies!

Part 2.1: Image Sharpening

We implemented unsharp masking to sharpen images by enhancing their high frequencies.

Part 2.2: Hybrid Images

By combining low frequencies of one image and high frequencies of another, we created hybrid images.

Part 2.3: Gaussian and Laplacian Stacks

We implemented Gaussian and Laplacian stacks for multi-resolution blending.

Part 2.4: Multiresolution Blending (The Oraple)

We blended an apple and an orange seamlessly using Laplacian stacks and Gaussian mask stacks.
